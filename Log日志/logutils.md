## **Log框架**

| 框架名称                                     | 功能描述                 |
| :--------------------------------------- | :------------------- |
| [Logger](https://github.com/orhanobut/logger) | 简单，漂亮，强大的Android日志工具 |
| [Hugo](https://github.com/JakeWharton/hugo) | 在调试版本上注解的触发方法进行日志记录  |
| [Timber](https://github.com/JakeWharton/timber) | 一个小的，可扩展的日志工具        |
